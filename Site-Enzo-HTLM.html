C'est quoi le HTML

Le HTML est un codage. Ses initiales signifient HyperText Mark-Up Language. C'est un Language dit de "marquage" (de "structuration" ou de "balises") celle-ci sont la por servir a formaliser l'ecriture d'un document avec des balises de formatage.Les balises servent a indiquer la manières dont doit etre présnenté le document et les liens qu'il établi avec d'autres documents.

Présentation

Le langage HTML permet notamment la lecture de documents sur Internet à partir de machines différentes, grâce au protocole HTTP, permettant d'accèder via le réseau à des documents repérés par une adresse unique, appelée URL. On appelle World Wide Web (noté WWW) ou tout simplement Web (mot anglais signifiant toile) la "toile virtuelle" formée par les différents documents (appelés « pages web ») liés entre-eux par des hyperliens. Les pages web sont généralement organisées autour d'une page d'accueil, jouant un point central dans la navigation à l'aide des liens hypertextes. Cet ensemble cohérent de pages web liées par des liens hypertextes et articulées autour d'une page d'accueil commune est appelée site web. Le Web est ainsi une énorme archive vivante composée d'une myriade de sites web proposant des pages web pouvant contenir du texte mis en forme, des images, des sons, des vidéo, etc.

  • Evolution du HTML

    • Durant la première motié des années 1990, avant l'apparition des tecnologies web comme le language Java Script(js),les feuilles de style en cascade(css) et le Document Object Model(Dom),l'évolution de HTML a fortement ralenti; 10 ans plus tard, HTML 4 reste utilisé dans les pages web. En 2008, la spécification de HTML5 est à l'étude et devient d'usage courant dans la seconde moitié des années 2010.

Le CSS

C'est quoi le CSS

Le Css est un style de language qui définit la présentation des documents HTML. Par exemple, CSS couvre les polices, les couleurs, les marges, les lignes, la hauteur, la largeur, les images d'arrières-plan, les positionements évolués et bien d'autres chose.Attendez de voir!

Présentation

Le CSS a été développé afin d'ajouter des informations graphiques à des documents XML et HTML.
Avec le développement de ce langage, le but du W3C est de nettoyer le langage HTLM de toutes les extensions graphiques ajoutées au cours de son histoire pour se concentrer sur les informations structurelles et sémantiques du document.
Le Langage XML étant exclusiment structurel, il était normal que le langage CSS soit concu pour aussi s'adapter sur ce langage.
Le principe est que les informations graphiques sont centralisées dans un fichier CSS qui pourra etre partagé entre les différentes pages d'un web (ou bien plusieurs sites web).
Ceci permet une réduction de la bande passante en factorisant toutes les informations graphiques d'un site dans un seul fichier.
